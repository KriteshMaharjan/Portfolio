# Portfolio

A personal portfolio website built with React and Vite showcasing projects, skills, and contact information.

**Tech stack:** React, Vite, ESLint

## Features
- **Home:** Hero section and intro
- **Navbar:** Responsive navigation
- **Projects:** Individual project pages/components
- **Skills:** Skillset display with categories
- **Contact:** Contact form / contact details
- **Back-to-top:** Smooth scroll helper

## Notes
- This project uses Vite for fast development and optimized production builds.
